# CLI Bitcoin Converter

A CLI convert Bitcoin to any currency provided.

This project was developed to learn about Javascript with Test Driven Development (TDD).

### Installing

Install node depencies

```
npm install
```

Build
```
npm run build
```

Link
```
npm link
```

Run
```
btc-converter --help
```
